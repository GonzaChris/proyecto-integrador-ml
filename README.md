# Proyecto Integrador - Módulo 6: Aplicaciones Avanzadas de Machine Learning y Deep Learning

## Descripción general

Este proyecto integra:
- Procesamiento de Lenguaje Natural (PLN) para analizar mensajes de clientes.
- Análisis de imágenes con redes neuronales convolucionales (CNN) para detectar emociones faciales.
- Aprendizaje por refuerzo para priorizar casos de soporte en función del riesgo emocional.

El objetivo es construir un sistema que, a partir de texto e imágenes, apoye la decisión de qué casos atender primero, considerando también aspectos éticos como privacidad, sesgos y equidad.

## Estructura del proyecto

- `data/`: conjuntos de datos (no se suben a GitHub).
  - `raw/`: datos originales.
  - `processed/`: datos procesados.
- `notebooks/`: Jupyter Notebooks con análisis, modelos y resultados.
- `src/`: código fuente reutilizable (funciones, clases, modelos).
- `reports/`: informe final (Word/PDF).
